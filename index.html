Para que serve o HTML

O HTML é utilizado para que os navegadores possam interpretar informações.

Mas, afinal o que é um HMTL?

O HTML é um arquivo de marcação. Textos delimitados por nomes que o navegador consegue ler.

Cada nome destes delimitadores tem uma função especifica

"O HTML oferece diversas vantagens que facilitam a criação e organização de páginas web." *

Estrutura do HTML

HTML consiste em três partes:

  1. Tag HTML
  2. Cabeçalho
  3. Corpo

Tag HTML

As tags HTML são escritas entre colchetes angulares < > e seguem uma estrutura específica.

Cabeçalho

Responsável por carregar antes de exibir ao usuário.

Corpo

Onde vão as informações que vão aparecer para o usuário.

O que são atributos das tags

De acordo com o site HTML Progressivo, atributos são comandos específicos que damos a uma tag para que ela faça o tal comando.
